# Desktop

In the **WEB** directory you can run: (*if necessary remove de host: "0.0.0"*)

1. `npm i`
2. `npm run dev`

![](web/src/assets/nlw-web.png)

# Mobile

In the **MOBILE** directory you can run:

1. `npm i`
2. `npx expo start`
